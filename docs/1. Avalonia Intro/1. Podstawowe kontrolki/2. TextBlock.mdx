# `<TextBlock>`
- TextBlock w Avalonia UI to lekka kontrolka do wyświetlania tekstu. Służy do prezentacji statycznych lub dynamicznych treści tekstowych.
- Główne cechy:
    - Nie obsługuje edycji – tylko do wyświetlania tekstu (w przeciwieństwie do TextBox).
    - Można zmieniać czcionkę, kolor, wyrównanie, zawijanie tekstu itp.
    - Lekka i wydajna, dobra do długich tekstów.


```xml
<StackPanel>
      <TextBlock Text="Witaj w Avalonia UI!"
                 Margin="10"
                 Padding="20"
                 />

      <TextBlock Text="Stylizowany tekst"
                 FontSize="20" Foreground="Red" TextAlignment="Center"/>
</StackPanel>
```
