# StackPanel
- Ustawia elementy w pionie (`Orientation="Vertical"`, domyślnie) lub poziomie (`Orientation="Horizontal"`).
- Elementy są rozmieszczane jeden po drugim w danym kierunku.

## Przykłady

### Orientacja pozioma:
```xml

<StackPanel Orientation="Horizontal" HorizontalAlignment="Center">
     <Border Width="200" Height="100" Margin="30"
             Background="LightBlue" BorderBrush="Black" BorderThickness="2"/>
     <Border Width="200" Height="100" Margin="30"
             Background="Yellow" BorderBrush="Black" BorderThickness="2"/>
     <Border Width="200" Height="100" Margin="30"
             Background="Fuchsia" BorderBrush="Black" BorderThickness="2"/>
</StackPanel>
```

### Orientacja pionowa:
```xml

<StackPanel Orientation="Vertical" HorizontalAlignment="Right">

    <Border Width="200" Height="100" Margin="30"
            Background="LightBlue" BorderBrush="Black" BorderThickness="2"/>

    <Border Width="200" Height="100" Margin="30"
            Background="Yellow" BorderBrush="Black" BorderThickness="2"/>

    <Border Width="200" Height="100" Margin="30"
            Background="Fuchsia" BorderBrush="Black" BorderThickness="2"/>
</StackPanel>
```

### Przeniesiemy prostokąty na dół okna:
`<StackPanel Orientation="Vertical" VerticalAlignment="Bottom" >`

### Przesuniemu na dół i w lewą stronę:
`<StackPanel Orientation="Vertical" VerticalAlignment="Bottom" HorizontalAlignment="Left" >`

# Grid
- Rozmieszcza elementy w siatce złożonej z wierszy i kolumn.
- Każdy element może być przypisany do określonego wiersza (`Grid.Row`) i kolumny (`Grid.Column`).
- Grid oferuje dużą elastyczność i możliwość tworzenia złożonych układów.

```xml
<Grid>
    <Grid.RowDefinitions>
        <RowDefinition Height="Auto" />
        <RowDefinition Height="*" />
        <RowDefinition Height="2*" />
        <RowDefinition Height="3*" />
    </Grid.RowDefinitions>

    <Grid.ColumnDefinitions>
        <ColumnDefinition Width="Auto" />
        <ColumnDefinition Width="*" />
        <ColumnDefinition Width="2*" />
    </Grid.ColumnDefinitions>

    <TextBlock Grid.Row="0" Grid.Column="0" Margin="10" Text="Label 1" />
    <Button Grid.Row="0" Grid.Column="1" Margin="10" Content="Button 1" />
    <TextBlock Grid.Row="1" Grid.Column="0" Margin="10" Text="Label 2" />
    <Button Grid.Row="1" Grid.Column="1" Margin="10" Content="Button 2" />
    <Rectangle Grid.Row="2" Grid.Column="0" Margin="10" Width="100" Height="50" Fill="Aqua"
                Stroke="Brown" StrokeThickness="3"/>
    <Rectangle Grid.Row="2" Grid.Column="1" Margin="10" Width="100" Height="50" Fill="Crimson"
                Stroke="Brown" StrokeThickness="3"/>
    <Rectangle Grid.Row="2" Grid.Column="2" Margin="10" Width="100" Height="50" Fill="Lime"
                Stroke="Brown" StrokeThickness="3"/>
    <Rectangle Grid.Row="3" Width="300" Height="50" Grid.ColumnSpan="2"
                Fill="Teal" Stroke="Fuchsia" StrokeThickness="4"/>
</Grid>
```
- `Auto`  - szerokość kolumny / wysokość wiersz będzie dopasowany do zawartości
- `*`     - dynamicznie dopasowuje wysokość/szerokość:
    - `2*` wysokość/szerokość zajmie dawa razy więcej przestrzeni niż `*`
    - `3*` ta wartość spowoduje, że wysokość/szerokość będzie trzy razy większa od `*`

# WrapPanel
Układa elementy w rzędzie i zawija je do nowego wiersza lub kolumny, jeśli nie mieszczą się w dostępnej przestrzeni.

```xml
<Window xmlns="https://github.com/avaloniaui"
        xmlns:x="http://schemas.microsoft.com/winfx/2006/xaml"
        xmlns:d="http://schemas.microsoft.com/expression/blend/2008"
        xmlns:mc="http://schemas.openxmlformats.org/markup-compatibility/2006"
        mc:Ignorable="d" d:DesignWidth="800" d:DesignHeight="450"
        x:Class="managerlayout.MainWindow"
        Title="managerlayout"
        Width="200"
        >

        <WrapPanel Orientation="Horizontal" ItemWidth="100" ItemHeight="50">
                <Button Content="Button 1" />
                <Button Content="Button 2" />
                <Button Content="Button 3" />
                <Button Content="Button 4" />
                <Button Content="Button 5" />
        </WrapPanel>
</Window>
```
